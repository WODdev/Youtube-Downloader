    # Youtube-Downloader
    **Command line based YouTube downloader**
    
    Supports mp3 & mp4

    **USAGE**
        install the zip file and unzip to your pc.
        Then cd the folder.
        Finally run this command line prompt: python yt.py [Lik] [mp3/mp4]
